**基于 wechat 做为即时通讯，使用 chatGPT 通过嵌入搜索方式实现私人领域模型问答**



快速使用

```shell
npm install 或 cnpm install
缺失的包再npm install xxx

运行
npm run start
python openai-support.py
```
